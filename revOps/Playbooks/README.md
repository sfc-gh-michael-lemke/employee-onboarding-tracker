# Playbooks Index

> Active onboarding playbooks generated by the Commission Onboarding Agent.
> One file per employee engagement, named `YYYY-MM-DD-Employee-Name-Playbook.md`.

---

## Active Playbooks

```dataview
TABLE employee, role, hire_date, status, gap
FROM "Playbooks"
WHERE type = "playbook"
SORT hire_date DESC
```

---

## Playbooks with Data Gaps

```dataview
TABLE employee, role, hire_date
FROM "Playbooks"
WHERE gap = true

```

---

## How to Generate a New Playbook

1. Open [[Agents/Commission-Onboarding-Agent|Commission-Onboarding-Agent.md]]
2. Copy the System Prompt and User Turn into your Cortex Agent
3. Fill in the employee details in the User Turn template
4. The agent will write the playbook to this folder automatically
