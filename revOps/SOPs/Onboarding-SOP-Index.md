## How to Import a New Document

1. Add the document to the Google Drive or Confluence space indexed by `DOCS_SOP_SEARCH`
2. Wait for the next scheduled refresh (weekly) or trigger a manual refresh
3. Add a row to the Core SOPs table above with the document title, source, and phase coverage
4. Link the document name to the relevant Phase note in [[Phases/]]
